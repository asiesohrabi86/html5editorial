# html5editorial

## English

This project is an attempt to recreate the [Editorial](https://html5up.net/editorial) template from HTML5 UP using **Bootstrap 5** and **jQuery**. The layout is responsive and works across all screen sizes.

### Features

- **Responsive Design:** Works seamlessly on all device sizes.
- **Modern Stack:** Built with Bootstrap 5 and jQuery for flexibility and compatibility.
- **Inspired by HTML5 UP:** Layout, style, and structure closely follow the original Editorial template.

### Sidebar Scroll Note

For screens wider than 1200px, I was unable to replicate the exact sidebar scroll behavior from the original template. Instead, I implemented a custom solution for the sidebar's scrolling. If you know how to make the sidebar scroll exactly like the original [Editorial](https://html5up.net/editorial) template, please let me know or contribute!

## فارسی

این پروژه تلاشی است برای پیاده‌سازی قالب [Editorial](https://html5up.net/editorial) از سایت HTML5 UP با استفاده از **Bootstrap 5** و **jQuery**. قالب کاملاً ریسپانسیو طراحی شده و برای همه‌ی اندازه‌های نمایشگر مناسب است.

### امکانات

- **طراحی واکنش‌گرا:** نمایش صحیح و زیبا در تمامی اندازه‌های نمایشگر
- **استفاده از تکنولوژی‌های به‌روز:** پیاده‌سازی با بوت‌استرپ ۵ و جی‌کوئری
- **الگو گرفته از HTML5 UP:** سعی شده ساختار و ظاهر قالب مشابه نسخه اصلی باشد

### نکته درباره اسکرول سایدبار

در سایزهای بالاتر از ۱۲۰۰ پیکسل، نتوانستم دقیقا رفتار اسکرول سایدبار در قالب اصلی را پیاده‌سازی کنم و مجبور شدم راه‌حل خودم را اجرا کنم. اگر کسی می‌داند چطور می‌توان دقیقا مانند سایت اصلی رفتار اسکرول سایدبار را پیاده کرد، خوشحال می‌شوم راهنمایی کند یا در پروژه مشارکت داشته باشد!
